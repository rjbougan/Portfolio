Project 1

This project takes an English word and translates it into Pig Latin. The project uses if and while loops to check if the entered text begins with vowel; if it does, it uses the .slice() method to create the translation. If the word starts with a consonant, it simply appends "ay" to the end of the string.

Pig Latin Translator
